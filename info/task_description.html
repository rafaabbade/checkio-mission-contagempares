Enunciado

Agora o Professor Thiago está ensinando seus alunos sobre sequências numéricas. Ele precisa de uma ferramenta que mostre aos alunos como contar de 2 em 2 até um limite especificado e armazenar esses números em uma lista.

Escreva uma função que receba um número inteiro n e use um loop while para criar uma lista dos números contando de 2 em 2,começando de 2 até n (inclusive). A função deve retornar a lista. Se n for menor que 2, a função deve retornar uma lista vazia.

Entrada: Um número inteiro n.

Saída: • Uma lista contendo todos os números pares entre 2 e n (inclusive), em ordem crescente. Se n for menor que 2, a lista deve estar vazia.
