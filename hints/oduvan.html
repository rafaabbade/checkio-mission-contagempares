Vamos dividir em partes. Primeiramente, vamos aprender a criar uma lista. Não se preocupe com detalhes, pois ainda vamos praticar muito o uso das listas.

Criar uma lista vazia é muito parecido com criar uma variável, basta escolher um nome e atribuir colchetes vazios a ela: 

lista = [] # cria uma lista vazia

Legal, agora tenho uma lista vazia. Como posso adicionar os números?

Para adicionar números à lista, utilizamos o comando .append(). O comando é usado conforme indicado abaixo: 

lista = [] # cria uma lista vazia
lista.append(1) # o nome da lista sempre vem antes do .append() e o que iremos adicionar a lista vem sempre entre parênteses.

O comando .append() adiciona um elemento no final da lista.

E o restante dos números?

Agora vamos aprender um novo laço de repetição: o while, ou enquanto.

O laço while é uma estrutura de repetição em Python que executa repetidamente um bloco de código enquanto uma determinada condição é verdadeira. A condição é verificada antes de cada execução do bloco de código. Se a condição for verdadeira, o bloco de código é executado. Se a condição for falsa, a execução do laço é interrompida.

Para visualizar um exemplo, assista a este vídeo.

Foi um bom exemplo, mas ainda estou com dúvidas sobre como escrever o código.

O while tem o funcionamento indicado abaixo. Faça alguns testes para verificar. 

i = 1 # usaremos a variável i como contador
while i <= n: # enquanto i é menor do que n, o código abaixo será executado
  print(i) # aqui estamos imprimindo um valor, mas devemos adicioná-lo à lista. Ainda bem que você já sabe fazer isso :)  
  i += 1 # somamos 1 ao i, para que o código não seja executado para sempre

Não se esqueça de retornar a lista!
